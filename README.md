# intent API

API layer for [Intent Calendar](https://intent-app.vercel.app/)


## Technologies used

- Firebase
- Docker
- AWS EC2
- AWS Route 53
- Nginx
- ASP.NET Core Web API
- Entity Framework
- PostgreSQL
